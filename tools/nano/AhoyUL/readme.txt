### The control and data interface of AhoyUL-app is the serial-IF (maybe a bit like tke the Arduino CUL modules for 868MHz with CC1101 tranceiver)
    # no NTP need, just unix epoch time sec increment
    # no display except serial-IF
    # very basic HM800 decoder in automode (other Inverter like HM400, HM600 or HM1500 easy to add in src)
    # possibly MQTT but actually no need yet

### The AhoyUL-app (in automode) starts to poll fast (5sec) after startup, then with regular interval (e.g. 120sec) and if still no response then with long interval (20min) during night.
# after one long poll-interval there there will be a fast poll period of 5 times (5 x 10sec, configurable in src)
